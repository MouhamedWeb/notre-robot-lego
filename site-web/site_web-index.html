Le Robot LEGO

Notre robot a été conçu à partir du kit LEGO Mindstorms/Spike. Il est capable de :

  • Suivre une ligne
  • Éviter les obstacles
  • Détecter plusieurs couleurs

Explication de la Programmation

Nous avons utilisé le langage C avec ROBOTC pour le labyrinthe, et Python avec Visual Studio Code pour le suiveur de piste.

Exemples de blocs utilisés :

  • Détection d'obstacles avec capteurs ultrason et Capteurs de contact
  • Instructions conditionnelles pour prendre des décisions
  • Contrôle des moteurs pour les mouvements complexes
  • Détection du piste et des différentes couleurs
Code C - Labyrinthe mission 1 Code Python - Labyrinthe mission 2 Code Python - Suiveur de piste mission 1 Code Python - Suiveur de piste mission 2
